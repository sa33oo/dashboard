# 動物病院向け 月次売上ダッシュボード v0.1

ブラウザでExcelをアップロードし、2026年4月の月次売上をその場で集計・可視化するローカル試作品です。

## セットアップ

```bash
npm install
npm run dev
```

## 画面機能

- 月次サマリー（売上合計・会計件数・平均会計単価・ユニークペット数）
- 決済別売上（支払/返金方法別の金額・件数）
- 日別売上
- 診療項目分析（診療項目名・診療カテゴリ）
- ペット分析（動物種別・品種・年齢）
- 検算画面（正解値との一致確認）

## 入力ファイル

- 売上データ（必須）
- 飼い主・ペット情報（任意だがペット分析で推奨）

サンプルは `data/sample/` 配下を使用してください。
